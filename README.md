## Description
VirtualMicroConsole is a project featuring a monochrome fantasy console that includes a sprite editor, a map editor, and a built-in library of sounds and music.
The goal of this project is to provide a coding environment that is as simple and minimalistic as possible. I originally created it mainly for people to have fun experimenting with it.
However, please note that this is a small project that I made without overthinking it. The code is quite messy (sorry about that), and there are probably many bugs.
For this reason, I included an error.txt file with the application, which is intended to describe the cause of an error in case of a crash or if the application does not work properly.
Please note that the current version is designed to run on Windows on. Also, I have included among the source files two projects named respectively PONG and MICRO RPG, which are small examples of projects that can be created using this application.
